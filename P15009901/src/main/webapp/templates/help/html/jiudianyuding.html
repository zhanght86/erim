  • 酒店频道预订须知
  • 第一条、目的
  • 统一规范酒店预订，减少酒店预订风险。
  • 第二条、适用对象
  • 酒店频道产品
  • 第三条、预订须知
  • 1、预订
  • 1.1 在线预订
  • 登陆佳旅联合旅游网酒店频道，按系统提示完成您的预订；为确保您的行程，在您填写预订单时，务必留下准确入住人姓名（请确保与所使用的有效证件上的姓名一致）及联系人手机号码。
  • 1.2 最晚预订时间
  • 1.2.1 建议您提前1－2天预订。但节假日、会展期间或旅游旺季最好提前较长时间预订，以防酒店客满。
  • 1.2.2 部分酒店有规定最晚预订时间，超过最晚预订时间不接受预订，具体以系统提示为准。
  • 1.3 最晚修改及取消时间
  • 预付及担保订单有规定的最晚修改及取消时间，超过此一时将可能给您造成损失。
  • 1.4 当您需要预订多间客房时，须提供不少于所预订客房数量的实际入住人的信息，入住人姓名需要不同， 以方便预订确认。
  • 1.5 如您已为相同入住人预定了入离时段相同（或部分重合）的客房，请不要重复预定。
  • 1.6 在您提交订单，接到确认信息后，表示您的预订成功，您可如期安排您的行程。
  • 2、费用说明
  • 2.1 网站前台呈现的酒店价格为酒店房费，不包含酒店其他费用及税收。
  • 2.2 网站价格可能因为酒店原因会有所变更，请以最终确认为准。
  • 2.3 您入住酒店的早餐与加床价格视各酒店规定有所不同。
  • 3、入住、延住与离店
  • 3.1 通常情况下，酒店的最早入住时间为14：00，最晚离店时间为正午12：00。
  • 3.1.1 若您提前入住，请与酒店协商，可能会遇到无空房的情况，需要您等待；
  • 3.1.2 若您推迟离店，需与酒店协商，酒店一般会加收一定的费用。
  • 3.2请您在预订单注明的最晚到店时间前到达酒店，如未能如期到达请及时与酒店联系，否则酒店将会取消您的订单。
  • 3.3 若您因行程变化取消了酒店订单，但之后再次需要预订时，价格将有可能会变更，请以最新确认的价格为准。
  • 4、您的权利和责任
  • 4.1 若订单已确认，您可如期入住所预订的酒店。
  • 4.2 若订单已确认，您在约定时间至预订酒店前台发现无房，请立即拨打电话通知佳旅联合，以便我们尽快帮您协调解决。
  • 4.3 您对您的订房亦负有相应的承诺，如由于您已预付或佳旅联合为您预付房款后而未入住所造成的损失亦将由您承担。如您在酒店入住时遇到困难，请致电佳旅联合，佳旅联合将为您提供必要的帮助。
  • 5、关于信用卡担保
  • 5.1 当旅游旺季、节假日期间或举办大型会展期间酒店房间紧张等情况下，部分酒店需要您提供信用卡担保。
  • 5.2 信用卡担保所支持的卡种以系统提示为准，并请您提供正确的信用卡信息，如信息不正确或通不过系统审核，则您将无法预订需要担保的酒店。
  • 5.3 需要信用卡担保的酒店，订单一经确认将不可取消或修改。因此请在填写担保订单前请慎重考虑您是否会修改行程，以免给您造成不必要的损失。
  • 5.4 如您未按订单时间或房间数量入住， 酒店将按国际惯例扣除相应的房费。
  • 6、 免责条款
  • 6.1 佳旅联合旅游网酒店产品页显示特定地点的酒店信息，但不代表建议去这些地方旅行或认为没有风险，对于去这些地方旅行造成的损害或损失佳旅联合旅游网不承担任何责任。
  • 6.2 佳旅联合旅游网对您入住酒店期间发生的意外不承担任何责任。建议您查看将要去旅行的国家政府发布的相关旅行公告，并采取必要的预防措施。同时建议您购买个人旅行保险。
  • 6.3 对于服务器错误、误导或重新导入传输、网络连接失败、传输或接受预订中断或计算机病毒或其他技术缺陷（人为的或技术的）造成的故障，佳旅联合旅游网不承担任何责任。
  • 第四条、施行时间
  • 本规范于即日起正式施行。
  • 第五条、附则
  • 本标准最终解释权归平台事业部酒店产品部所有。
